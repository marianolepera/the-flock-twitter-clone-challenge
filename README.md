# The Flock — Twitter Clone Challenge

Monorepo con scaffolding inicial: **NestJS** (`backend/`) y **React + Vite** (`frontend/`).

## Estructura

```
├── backend/    # NestJS API
├── frontend/   # React + Vite
└── README.md
```

## Prerrequisitos

- Node.js 20+ (recomendado 24 LTS)
- npm 10+
- Git

## Desarrollo local

```bash
# Backend (http://localhost:3000)
cd backend
npm install
npm run start:dev

# Frontend (http://localhost:5173)
cd frontend
npm install
npm run dev
```
